# CISC275-honors-discussion-10

Open `index.html` in a browser to view the streamlined file explorer demo.

## GitHub Pages deployment

This repository includes a GitHub Actions workflow at `.github/workflows/pages.yml` that deploys this static site to GitHub Pages whenever you push to `main`.

After merging to `main`, your site will be available at:

`https://jstepnow06.github.io/CISC275-honors-discussion-10/`

### One-time repository setting

In GitHub, go to **Settings → Pages** and set **Source** to **GitHub Actions**.
